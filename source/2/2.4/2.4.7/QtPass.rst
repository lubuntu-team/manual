Chapter 2.4.7 QtPass
====================

QtPass is the default password manager for Lubuntu. 

Usage
------

On first launch you will be asked to generate a key to encrypt your passwords. The first field to generate the encryption key is to enter your emial in the :guilabel:`Email` field.

Version
-------
Lubuntu ships with version 1.2.3 of QtPass.

How to Launch
-------------
To launch Qtpass from the menu run :menuselection:`Accessories --> QtPass` or from the command line run

.. code:: 

    qtpass
